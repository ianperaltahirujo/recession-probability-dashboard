What's inside

Built like a production pipeline

Every layer reflects how data science works at a professional level: automated ingestion, tracked experiments, calibrated models, and a live deployed interface.

Live FRED Data Pipeline
Pulls 7 macroeconomic series from the Federal Reserve API weekly. Resamples mixed-frequency data to monthly and caches locally to avoid redundant API calls.
fredapi · pandas
Feature Engineering
19 recession-signal features including yield curve inversion flags, MoM/YoY rate-of-change transforms, and 3-month rolling averages across all indicators.
scikit-learn · numpy
Calibrated Gradient Boosting
GradientBoostingClassifier wrapped with isotonic calibration via CalibratedClassifierCV for well-calibrated probability outputs. CV AUC-ROC: 0.9977.
gradient boosting · calibration
MLflow Experiment Tracking
Every training run logs hyperparameters, CV metrics, and model artifacts. Logistic regression baseline included for a meaningful comparison across runs.
mlflow · model registry
Prefect Orchestration
Weekly flow chains fetch, engineer, and score tasks with automatic retries and exponential backoff. Task dependencies enforced so bad data never cascades downstream.
prefect · scheduling
Live Deployed Dashboard
Interactive Streamlit dashboard on Streamlit Community Cloud. Log-scale probability chart, indicator readings, and full methodology. Pre-computed scores served at request time.
streamlit · altair
Architecture

How the pipeline works

Five stages run in sequence every week. If any stage fails, downstream tasks are blocked automatically by Prefect.

01
Fetch FRED Data
Pulls T10Y2Y, UNRATE, ICSA, INDPRO, UMCSENT, BAA10Y, PAYEMS, and USREC from the St. Louis Fed API. Resamples weekly series to monthly frequency.
fredapi
02
Engineer Features
Computes 19 features: rolling means, MoM/YoY transforms, yield curve inversion flag, and jobless claims trend. Drops NaN rows from rolling warmup period.
pandas · numpy
03
Train & Track
Trains calibrated gradient booster and logistic regression baseline using 5-fold time-series CV. All params and metrics logged to MLflow. Best model saved.
mlflow · sklearn
04
Score Current Conditions
Loads best model and scaler, scores all 421 months of feature data, writes scores CSV and current snapshot JSON for dashboard consumption.
joblib
05
Serve Dashboard
Streamlit reads pre-computed scores and snapshot. No model inference at request time. Cached for 1 hour. Deployed on Streamlit Community Cloud.
streamlit

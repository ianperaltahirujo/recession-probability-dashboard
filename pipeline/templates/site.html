What's inside

Built like a real pipeline, not a demo

Every layer is a real, separately runnable module not one script that does everything.

Live FRED Data Pipeline
Pulls 7 macroeconomic series from the Federal Reserve API weekly and resamples mixed-frequency data to monthly.
fredapi · pandas
Feature Engineering
19 recession-signal features: yield curve inversion, MoM/YoY transforms, rolling averages across every indicator.
scikit-learn · numpy
Calibrated Gradient Boosting
Isotonic calibration via CalibratedClassifierCV for well-calibrated probabilities, not near-binary scores. CV AUC-ROC {{ cv_auc }}.
gradient boosting
MLflow Experiment Tracking
Every training run logs hyperparameters and CV metrics against a logistic regression baseline for real comparison.
mlflow
Local Orchestration
A Prefect flow chains fetch, engineer and score locally with retries. Production runs the same stages directly from GitHub Actions; Prefect isn't in the deploy path.
prefect · local only
Weekly Static Regeneration
Every Monday, GitHub Actions reruns the pipeline and rebuilds this exact page from the fresh output. No server, no cold start.
github actions · pages

Architecture

How the pipeline works

Four stages run automatically every Monday. Retraining is a deliberate manual step, not part of the weekly cadence.

01
Fetch FRED Data
Pulls T10Y2Y, UNRATE, ICSA, INDPRO, UMCSENT, BAA10Y, PAYEMS, and USREC. Resamples weekly series to monthly.
Weekly
02
Engineer Features
Computes 19 features: rolling means, MoM/YoY transforms, yield curve inversion flag.
Weekly
03
Train & Calibrate
Trains the calibrated gradient booster against a logistic regression baseline. Run by hand when retraining; the resulting model is committed manually.
Manual
04
Score Current Conditions
Loads the committed model and scaler, scores all {{ n_months }} months, writes scores and a snapshot JSON.
Weekly
05
Generate Static Site
Renders this page from the fresh scores and snapshot, and commits it back to the repo for GitHub Pages to serve.
Weekly
